# IDX Dividend Tracker

Screen saham IDX dengan **dividend yield > 5%** dan analisis kelayakan dividen.

## Rekomendasi

| | Rekomendasi | Kondisi |
|---|---|---|
| 💰 | **Beli Dividen** | Yield >5%, payout sehat, dividen tumbuh |
| ✅ | **Tahan** | Yield >5%, stabil, tidak ada red flag |
| 📈 | **Alih Capital Gain** | Dividen menyusut, lebih baik ambil gain |
| ⛔ | **Hindari** | Payout ratio >90% atau dividend trap |
| 👀 | **Pantau** | Yield <5% atau perlu konfirmasi |

## Tips Timing

- **Beli sebelum ex-dividend date** untuk dapat dividen — idealnya 2-4 minggu sebelumnya
- **Waspadai "dividend capture"** — harga sering turun setelah ex-div date
- **Cek payout ratio** — >90% = tidak sustainable, kemungkinan dipotong
- **Yield >20%** = kemungkinan dividend trap — harga sudah jatuh karena masalah fundamental

Built with Python + yfinance + TradingView Scanner.
